Импорты

Импорты помещаются в самое начало файла и сортируются в следующем порядке:

  1. Импорты из стандартной библиотеки

  2. Импорты сторонних библиотек

  3. Импорты ваших модулей

При этом каждый блок сортируется от более общих импортов к более узким. Блоки сортируются по алфавиту.

Хорошо

import os from functools import cache import module import module.submodule import special from special import SomeClass

Плохо

import os, sys

Каждый импорт должен быть на отдельной строке.

Хорошо

import os import sys

Плохо

import os, sys

Никогда не используйте from module import *. Так вы не только засоряете глобальное пространство имен ненужными именами, но еще и делаете ваш код непонятным для других программистов.
