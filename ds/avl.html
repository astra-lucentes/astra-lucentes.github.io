AVL дерево

AVL дерево – сбалансированное двоичное дерево поиска. Балансировка производится по высоте: у каждого узла высоты левого и правого поддеревьев различаются не более чем на 111.

AVL деревья названы в честь их изобретателей, Г. М. Адельсона-Вельского и Е. М. Ландиса.

Оценим высоту AVL дерева. В дереве высоты hhh минимальное количество вершин Fh+2−1=Θ(φh)F_{h+2}-1 = \Theta(\phi^h)Fh+2​−1=Θ(φh). Значит высота AVL дерева hhh равна log⁡φn=O(log⁡n)\log_\phi n = O(\log n)logφ​n=O(logn).

Балансировка

После операций добавления или удаления дерево может перестать быть сбалансированным. Для восстановления свойства сбалансированности есть операции поворота и большого поворота.

Для контроля баланса определим balance factor – разницу между высотами левого и правого поддеревьев.

bf⁡v=height⁡(v.left)−height⁡(v.right)\bf v = \height(v.\text{left}) - \height(v.\text{right})bfv=height(v.left)−height(v.right)

Поворот

Обычный поворот относительно вершины выравнивает высоты её поддеревьев. После поворота высота дерева не увеличивается, то есть уменьшается на 111 или остается неизменной.

Если bf⁡A=2\bf A = 2bfA=2, то осуществляем правый поворот. Если bf⁡A=−2\bf A = -2bfA=−2, то левый поворот.

function rotate_right(node a): node b = a.left a.left = b.right b.right = a height adjustment a height adjustment b
function rotate_left(node a): node b = a.right a.right = b.left b.left = a height adjustment a height adjustment b

Большой поворот

В результате обычного поворота balance factor у вершины B может стать 222 или −2-2−2, и дерево снова будет разбалансированным. Эту проблему решают большие повороты влево и вправо, которые состоят из двух обычных поворотов: у вершины и у ее ребёнка.

function big_rotate_left(node a): rotate_right(a.right) rotate_left(a)
function big_rotate_right(node a): rotate_left(a.left) rotate_right(a)
