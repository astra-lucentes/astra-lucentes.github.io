Дерево Фенвика

Дерево Фенвика (binary indexed tree) – структура данных, позволяющая за O(log⁡n)O(\log n)O(logn) выполнять над массивом два типа операций

  1. Находить значение ассоциативной, коммутативной и обратимой операции на отрезке массива.

  2. Изменить один элемент массива.

Занимает минимальную память Θ(n)\Theta(n)Θ(n) (столько же, сколько сам массив). Строится за время O(nlog⁡n)O(n \log n)O(nlogn).

Построение дерева

Можно построить в лоб по определению за O(n2)O(n^2)O(n2), а можно выполнить nnn раз функцию modify.

В начале ttt состоит из единичных элементов относительно операции ∘\circ∘. Для каждого элемента массива aaa мы выполняем операцию modify, тем самым строим дерево за O(nlog⁡n)O(n \log n)O(nlogn).
