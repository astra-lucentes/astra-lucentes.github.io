Монотонный стек

Монотонный стек (стек рекордов) – стек, в котором каждый элемент больше (меньше) всех находящихся ниже.

Применение

Ближайшее большее слева

Дан массив aaa, состоящий из nnn целых чисел. Нужно для каждого элемента найти ближайший слева элемент, который его больший.

Можно создать убывающий монотонный стек. Проходимся по всем элементам массива aaa по порядку. Вытаскиваем из стека все элементы, меньшие текущего. Если стек после этого не опустел, то ответ для текущий позиции – верхний элемент стека. После текущий элемент добавляем в стек.
