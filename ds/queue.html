Очередь

Очередь – набор элементов, организованных по принципу FIFO (первым вошел – первым вышел). Работает как реальная человеческая очередь, где вставший раньше человек доберется до конца раньше.

Очередь поддерживает операцию добавления элемента в конец и извлечения элемента с начала за O(1)O(1)O(1).

Реализация циклической очереди на массиве

Очередь, способную вместить не более nnn элементов, можно реализовать на статическом массиве.

const int n type queue: array elements[n] int head int tail

Добавление элемента происходит в хвост, и хвост перемешается назад.

method push(self, value): assert self.size < n self.tail = (self.tail + 1) % n self.elements[self.tail] = value

При удалении элемента достаточно просто переместить голову.

method pop(self): assert self.size > 0 x = self.element[self.head] self.head = (self.head + 1) % n return x

Реализация на связном списке

Можно реализовать очередь на связном списке напрямую.

Нам нужен сам связный список и две ссылки на начало очереди head и конец очереди tail. Отдельно сохраняем длину очереди. Получаем следующую структуру

struct node: value node next type queue: node head node tail int size constructor(self): self.head = self.tail = none self.size = 0

Добавление элемента

Для добавления достаточно создать новый узел и перезаписать ссылки.

method push(self, value): tail = self.tail self.tail = node(value, none) if self.size == 0: self.head = self.tail else: tail.next = self.tail self.size++

Удаление элемента

Для удаления достаточно перезаписать ссылки и удалить узел.

method pop(self): assert self.size > 0 self.size-- head = self.head self.head = self.head.next return head
