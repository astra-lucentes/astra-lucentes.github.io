Стек

Стек – набор элементов, организованных по принципу LIFO (последним вошел – первым вышел). Можно представить стек как стопку тарелок: класть новую тарелку можно только на верх стопки и забирать тарелки можно только с верху стопки.

Стек поддерживает операцию добавления элемента на верх и извлечения элемента с верху за O(1)O(1)O(1).

Можно реализовать на основе связного списка или динамического массива (вставка на верх стека – добавление в конец массива, извлечение из стека – удаление с конца массива).

Реализация

type stack: private array a method push(self, value): self.a.append(value) method peek(self): return self.a[-1] method pop(self): top = self.peek() delete self.a[-1] return top
