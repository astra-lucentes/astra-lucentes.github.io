Хеш-функция FNV

Fowler-Noll-Vo – безумно простая некриптографическая хеш-функция.

Для алгоритма выбирается базис смещения offset_basis – начальное значение хеша и простое число prime, на которое будет постоянно умножаться хеш. Начальное значение не может быть 000, иначе все нулевые последовательности хешируются в 000.

const int prime, offset_basis function fnv(bytes data) -> unsigned 64-bit int: unsigned 64-bit int hash = offset_basis for each b in data: hash *= prime hash ^= b return hash
