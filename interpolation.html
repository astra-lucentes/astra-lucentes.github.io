Интерполяция

Интерполяция – нахождение функции по заданным значениям в некоторых точках (или функции, которая в заданных точках близка к заданным значениям).

Набор заданных точек называется узлами интерполяции.

Интерполяция точным многочленом

Интерполяционный многочлен Лагранжа

Многочлен степени nnn, проходящий через точки (x0,y0), (x1,y1), (x2,y2), …,(xn,yn)(x_0, y_0),\, (x_1, y_1),\, (x_2, y_2),\,\dots, (x_n, y_n)(x0​,y0​),(x1​,y1​),(x2​,y2​),…,(xn​,yn​)

f(x)=∑i=0nyi⋅∏j=0i≠jn racx−xjxi−xjf(x) = \sum\limits_{i=0}^n y_i \cdot \prod_{\substack{j=0\\i\neq j}}^n \ rac{x-x_j}{x_i-x_j}f(x)=i=0∑n​yi​⋅j=0i=j​∏n​ racx−xj​xi​−xj​

В точке xix_ixi​ значение многочлена f(xi)f(x_i)f(xi​) действительно равно yiy_iyi​, ведь для iii-го слагаемого внутреннее произведение равно 111, а для остальных 000.

Предположим, что есть два подходящих многочлена A(x)A(x)A(x) и B(x)B(x)B(x). Тогда в точках x0,x1,…xnx_0, x_1, \dots x_nx0​,x1​,…xn​ многочлен A(x)−B(x)A(x) - B(x)A(x)−B(x) равен 000, то есть у него n+1n+1n+1 корней.
Значит deg⁡(A+B)=n+1\deg (A+B) = n+1deg(A+B)=n+1, но deg⁡A=deg⁡B=n\deg A = \deg B = ndegA=degB=n. Противоречие.

def interpolation_lagrange(points: list[tuple[float, float]], z: float) -> float: n = len(points) x, y = zip(*points) s = 0 for i in range(n): p = 1 for j in range(n): if i == j: continue p *= (z - x[j]) / (x[i] - x[j]) s += y[i] * p return s
