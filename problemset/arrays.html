Массивы и списки. Задания

Предпочтительнее решать задачи на C. Решения оформляйте в виде функций, а не через консоль.

Код должен быть простым и понятным. Правильно именуйте переменные, следуйте правилам оформления кода. За несоблюдение буду снижать оценки и заставлять переделать.

Динамический массив

Реализуйте динамический массив на C с помощью структуры и функций.

typedef int T; typedef struct array { T* head; int size; int capacity; } array;

Нужно реализовать следующие функции

  • get(array* a, int i) – получить значение по индексу i

  • set(array* a, int i, T value) – изменить значение по индексу i на value

  • append(array* a, T value) – вставка в конец массива

  • delete(array* a) – удаление элемента с конца массива

  • insert(array* a, int i, T value) – вставить элемент на место i

  • remove(array* a, int i) – удалить элемент с индексом i

Удалить дубликаты

Дан отсортированный односвязный список (первый его узел). Необходимо удалить все дублирующиеся элементы, оставив только уникальные.

Например, 2 -> 2 -> 3 -> 4 -> 4 -> 4 -> 5 превращаем в 2 -> 3 -> 4 -> 5

Узел связного списка реализован как класс.

class node: def __init__(self, value=None, next=None): self.value = value self.next = next def __str__(self): return f"{self.value} -> {self.next}"

Решение - функция, которая принимает первый узел списка и возвращает первый узел результирующего списка.

def solution(a: node) -> node: ...

Для нашего примера a = node(2, node(2, node(3, node(4, node(4, node(4, node(5))))))) функция должна вернуть node(2, node(3, node(4, node(5))))

Развернуть список

Дан связный список (задан так же, как в предыдущей задаче). Необходимо его развернуть. Функция должна вернуть узел (указатель на узел) связного списка, который состоит из всех элементов исходного, расположенных в обратном порядке.

def revere(n: node) -> node: ...

Сумма двух

Дан массив чисел a и число target. Найдите в массиве a два числа, которые в сумме дают target. Верните индексы этих чисел.

Если таких чисел нет, то вызовете исключение ValueError. Если их несколько, верните любое.

def two_sum(a: list[int], target: int) -> tuple(int, int): ...

Например

a = [2, 7, 8, 3]; target = 9 -> (0, 1) a = [1, 2, 8, 0]; target = 2 -> (1, 3) a = [1, 1]; target = 4 -> ValueError

Гидрометцентр

Дан список из температур по дням. Верните список answer, в которомanswer[i] - количество дней, которые надо подождать, чтобы послеi-го дня температура стала больше.

def hmc(t: list[int]) -> list[int]: ...

Например,

t = [12, 14, 21, 17, 86, 15, 3] answer = [1, 1, 2, 1, 0, 0, 0]

Поворот

Дана матрица (двумерный массив) n×nn \times nn×n, состоящая из чисел. Поверните матрицу на 90 градусов по часовой стрелке, не создавая дополнительных массивов.

def rotate(matrix: list[list[int]]) -> list[list[int]]: ...

Например,

matrix = [[1,2,3], [4,5,6], [7,8,9]] answer = [[7,4,1], [8,5,2], [9,6,3]]

Очередь за билетами

В очереди стоит nnn человек. Эту очередь характеризует массив, каждый элемент массива говорит, сколько билетов хочет купить человек, который в очереди стоит на том же месте, что и в массиве.

Каждую секунду человек, стоящий в начале очереди (чему соответствует начало списка) покупает один билет, и, если ему еще нужно купить билеты, встает в конец очереди.

Сколько времени простоял в очереди человек, который изначально имел индекс kkk (здесь индексация с 0)?

def ticket_queue(queue: list[int], k: int) -> int: ...

Например, функция ticket_queue([2, 3, 2], 2) должна вернуть 666.

[ 2 , 3 , 2 ] 0 секунд [ 3 , 2 , 1 ] 1 секунда [ 2 , 1 , 2 ] 2 секунды [ 1 , 2 , 1 ] 3 секунды [ 2 , 1 ] 4 секунды [ 1 , 1 ] 5 секунд [ 1 ] 6 секунда, человек вышел из очереди
