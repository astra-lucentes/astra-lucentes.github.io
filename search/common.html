Поиск часто встречающегося элемента

Мажорирующий элемент

Мажорирующим элементом коллекции длины nnn называется элемент, встречающийся в ней более n2\frac{n}{2}2n​ раз.

Будем объединять разные элементы в пары и считать количество одинаковых. Заведем счетчик элементов без пары. Будем идти по массиву и запоминать элемент, для которого еще не нашлось пары. При встрече такого же элемента увеличиваем счетчик, иначе — уменьшаем. Если все элементы уже имеют пару, то говорим, что у текущего элемента пары нет.

function search_majority(collection a): int count = 0 candidate = none for x in a: if count == 0: candidate = x count++ else: if x == candidate: count++ else: count-- return candidate

Мы одн раз проходимся по всем элементам коллекции. Временная сложность алгоритма O(n)O(n)O(n).

Общий случай

Общая задача. Дана коллекция из nnn элементов. Надо найти элементы, которые встречаются более nk\frac{n}{k}kn​ раз.

Воспользуемся той же идеей, но элементы будем объединять не в пары, а в группы по kkk. Мы будем хранить все элементы, для которых еще не найдена группа, для каждого элемента заведем счетчик – количество таких элементов.

Мы одн раз проходимся по всем элементам коллекции. Временная сложность алгоритма O(n)O(n)O(n).
