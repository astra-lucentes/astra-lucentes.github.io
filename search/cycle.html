Поиск цикла эндоморфизма

Пусть дан эндоморфизм f ⁣:S→Sf \colon S \to Sf:S→S и стартовая точка x0x_0x0​. Задача нахождения цикла эндоморфизма состоит в нахождении таких минимальных μ\muμ (длина хвоста) и λ\lambdaλ (длина цикла), что fμ(x0)=fμ+λ(x0)f^\mu(x_0) = f^{\mu+\lambda}(x_0)fμ(x0​)=fμ+λ(x0​).

Алгоритм Флойда нахождения цикла.

  1. Создать два указателя: быстрый и медленный.

  2. Пока указатели не сравняются, перемещать медленный указатель на один шаг, а быстрый на два.

  3. Поместить медленный указатель в начало. Завести счетчик μ\muμ длины хвоста цикла.

  4. Пока указатели не сравняются, перемещать каждый из них на один шаг и увеличивать μ\muμ на 111.

  5. Заводим счетчик λ\lambdaλ длины цикла. Сразу установить в 111.

  6. Оба указателя сейчас в начальной точке цикла. Перемещать один из указателей, каждый шаг увеличивать счетчик на 111, пока указатели снова не сравняются.

function floyd(f, x₀): slow = f(x₀) fast = f(f(x₀)) while slow != fast: slow = f(slow) fast = f(f(fast)) μ = 0 slow = x₀ while slow != fast: slow = f(slow) fast = f(fast) μ += 1 λ = 1 fast = f(slow) while slow != fast: fast = f(fast) λ += 1 return μ, λ

Этот алгоритм работает за время O(μ+λ)O(\mu + \lambda)O(μ+λ) и константную память.
