Экспоненциальный поиск

Экспоненциальный поиск – быстрый поиск, работающий в безумно больших или бесконечных отсортированных последовательностях. Применяется для определения диапазона, в котором может находится искомый элемент.

Выполняется за время O(log⁡i)O(\log i)O(logi), где iii – индекс искомого элемента.

Алгоритм экспоненциального поиска.

  1. Установить правую границу диапазона поиска в 111.

  2. Пока элемент на границе меньше искомого, увеличить границу в 222 раза.

  3. Искомый элемент находится на индексах от половины границы до границы. Запустить бинарный (или любой другой) поиск для этого диапазона.

function exponential_search(sequence a, target): border = 1 while border < length of a and a[border] < target: border *= 2 return search(a[border/2 : border], target)
