Сортировка пузырьком

Сортировка пузырьком – квадратичная устойчивая сортировка.

Мы сравниваем соседние элементы, и, если порядок в паре неверный, то элементы меняют местами. За каждый проход по массиву как минимум один элемент встает на свое место, поэтому необходимо совершить не более n−1n-1n−1 проходов.

Временная сложность реализации в лоб Θ(n2)\Theta(n^2)Θ(n2), а пространственная O(1)O(1)O(1).

function bubble_sort(array a): for k = 0, k < n: for i = 0, i < n-1: if a[i] > a[i+1]: swap a[i], a[i+1]

После kkk-той итерации kkk элементов стоят на своих местах, поэтому во внутреннем цикле нет необходимости по ним проходиться.

Количество операций уменьшилось, но асимптотически время все равно Θ(n2)\Theta(n^2)Θ(n2).

function bubble_sort(array a): for k = 0, k < n: for i = 0, i < n - k - 1: if a[i] > a[i+1]: swap a[i], a[i+1]

Если во внутреннем цикле не было сделано ни одного обмена, то список уже отсортирован и больше ничего делать не надо.

function bubble_sort(array a): for k = 0, k < n: is_swapped = false for i = 0, i < n - k - 1: if a[i] > a[i+1]: swap a[i], a[i+1] is_swapped = true if not is_swapped: break

Таким образом временная сложность в лучшем случае составит Ω(n)\Omega(n)Ω(n) вместо Ω(n2)\Omega(n^2)Ω(n2), и итоговая временная сложность алгоритма улучшится до O(n2)∩Ω(n)O(n^2) \sect \Omega(n)O(n2)∩Ω(n).
