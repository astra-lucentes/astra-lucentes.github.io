Сортировка подсчетом

Сортировка подсчетом (Counting sort) – сортировка, позволяющая за линейное время сортировать массив, элементы которого принадлежат небольшому множеству.

Работает за время Θ(n)\Theta(n)Θ(n) и тратит память Θ(k)\Theta(k)Θ(k), где kkk – количество элементов в множестве значений.

Мы можем подсчитать, сколько раз каждый элемент встречается в данном массиве, а затем вернуть элементы в нужном порядке по столько раз, сколько нужно.

Для подсчета мы заводим счетчик – массив, в котором хранятся целые числа. Индекс массива будет являться подсчитываемым элементом, а значение по этому индексу – сколько раз этот элемент встречался в массиве. В начале все элементы счетчика равны 000.

function counting_sort(list a): array counter[k] = [0, ..., 0] for each x in a: counter[x]++ array result = [] for each n with index i in counter: repeat n: result.append(i) return result
