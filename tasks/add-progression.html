Добавить прогрессию

Medium

Дан массив aaa, состоящий из nnn целых чисел. Необходимо произвести qqq операций добавления прогрессии на отрезке и вернуть получившийся массив.

Операция – добавление прогрессии с начальным членом sss и разностью ddd на отрезке массива [l,r][l, r][l,r], то есть

К элементу ala_lal​ добавляется sss,
к al+1a_{l+1}al+1​ добавляется s+ds+ds+d,
к al+2a_{l+2}al+2​ добавляется s+2ds+2ds+2d,
к al+3a_{l+3}al+3​ добавляется s+3ds+3ds+3d,
и так далее
к ara_rar​ добавляется s+(r−l)⋅ds + (r - l) \cdot ds+(r−l)⋅d.

Напишите функцию, которая принимает начальный массив aaa и массив запросов. Каждый запрос – это кортеж из четырех чисел (l, r, s, d), означающий, что надо добавить прогрессию с начальным членом sss и разностью ddd к отрезку[l,r][l, r][l,r] массива aaa.

function array add_progression(array a, array[tuple] q): return массив после выполнения запросов q
