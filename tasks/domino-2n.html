Домино 2 × N

Найдите количество всевозможных разбиений прямоугольника 2×n2 \times n2×n на доминошки 1×21 \times 21×2

Решение

Обозначим за f(n)f(n)f(n) количество разбиений прямоугольника 2×n2 \times n2×n на доминошки 1×21 \times 21×2. Ясно, что f(1)=1f(1) = 1f(1)=1 и f(2)=2f(2) = 2f(2)=2. Попробуем выразить f(n)f(n)f(n) через предыдущие значения.

Для какого-то разбиения большого прямоугольника существует ровно два варианта расположения первых доминошек. Отсюда получаем соотношение

f(n)=f(n−1)+f(n−2)f(n) = f(n-1) + f(n-2)f(n)=f(n−1)+f(n−2)

Получаем, что f(n)=Fn+1f(n) = F_{n+1}f(n)=Fn+1​, а это n+1n+1n+1 число Фибоначчи.

Решение в лоб через рекурсию. Его сложность Θ(φn)\Theta(\varphi^n)Θ(φn) времени и O(1)O(1)O(1) памяти.

def f(n): if n <= 2: return n return f(n-1) + f(n-2)

Применим динамическое программирование. Заведем массив f, в котором будем хранить ранее вычисленные значения f(n)f(n)f(n). Таким образом, мы реализуем кеширование и избавимся от рекурсии.

def solution(n): f = [0] * (n+1) f[0] = 1 f[1] = 1 for i in range(2, n+1): f[i] = f[i-1] + f[i-2] return f[n]

Это решение работает за O(n)O(n)O(n) времени и памяти. Однако, мы на самом деле используем только три ячейки: текущий, предыдущий и предпредыдущий элементы. Поэтому в этом решении можно оптимизировать память.

def solution(n): a, b = 1, 1 for _ in range(2, n+1): a, b = b, a + b return b

Оптимизированное решение работает за O(n)O(n)O(n) времени и O(1)O(1)O(1) памяти.

Ответ на задачу – правый нижний элемент матрицы AnA^nAn. А степень матрицы находится за O(log⁡n)O(\log n)O(logn).

def matmul(a, b): return ( (a[0][0]*b[0][0] + a[0][1]*b[1][0], a[0][0]*b[0][1] + a[0][1]*b[1][1]), (a[1][0]*b[0][0] + a[1][1]*b[1][0], a[1][0]*b[0][1] + a[1][1]*b[1][1]) ) def fastpow(a, n): result = ((1, 0), (0, 1)) while n: if n & 1: result = matmul(result, a) a = matmul(a, a) n >>= 1 return result def solution(n): a = ((0, 1), (1, 1)) return fastpow(a, n)[1][1]
