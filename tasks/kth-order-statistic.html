kkk-я порядковая статистика

Medium

kkk-я порядковая статистика массива – это kkk-й по величине элемент этого массива, то есть тот, что при сортировке этого массива встает на kkk-е место.

Напишите функцию, которая ищет kkk-ю порядковую статистику заданного массива.

function int order_statistic(int k, array a): return k-я порядковая статистика массива a

Система оценивания

Детерминированное решение за O(n)O(n)O(n).
Недетерминированное решение за O(n)O(n)O(n).
Решение за псевдолинию или за O(nlog⁡log⁡n)O(n \log \log n)O(nloglogn).
Оптимизированная сортировка с выходом.
Сортировка и взятие kkk-го элемента.
