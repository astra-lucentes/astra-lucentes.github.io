Ближайшие точки

Easy

Перед вами обычная двумерная плоскость Z2\ZZ^2Z2 (как бесконечный клеточный лист). Вы выполняете два действия на этой плоскости:

  1. Рисуете новую точку (x,y)(x, y)(x,y) на плоскости

  2. Стираете самую близкую к центру точку.

Напишите класс, имеющий два метода: реализации этих действий. Метод draw_dot(int x, int y) должен добавить на плоскость точку (x,y)(x, y)(x,y). Метод erase_dot() должен стирать ближайшую к центру точку (удалять ее с плоскости) и возвращать координаты этой точки.

class Plane: method draw_dot(self, int x, int y): добавить точку (x, y) method erase_dot(self) -> (int, int): удалить с плоскости ближайшую к центру точку return координаты (x, y) удаленной точки
