ГлавнаяЗадачи

Отрезки запросов

Hard

Дан массив aaa, состоящий из nnn целых чисел. Дан массив операций добавления числа ddd ко всем элементам отрезка [l,r][l, r][l,r] массива aaa. Дан массив запросов. Каждый запрос означает выполнение всех операций с номерами из отрезка [L,R][L, R][L,R].

Напишите функцию, выполняющую все запросы и возвращающую результирующий массив.

function array query_segments(array a, array[tuple] operations, array[tuple] queries): return массив после выполнения всех запросов queries

Элемент массива operations – кортеж (d, l, r). Этот кортеж означает операцию добавления ddd ко всем элементам из отрезка [l,r][l, r][l,r] массива aaa.

Элемент массива queries – кортеж (L, R). Этот кортеж означает выполнение всех операций с номерами от LLL до RRR, то есть выполнение всех операций из отрезка [L,R][L, R][L,R] массива operations.
