Smallest in infinity set

Easy

У вас есть множество, содержащее все натуральные числа 1,2,3,…1, 2, 3, \dots1,2,3,…. Вы хотите выполнять три операции

  1. Удалить и вернуть наименьшее число в этом множестве

  2. Заново добавить число в множество

Напишите класс, представляющий это множество.

class InfinitySet: method pop_min(self) -> int: удалить наименьшее число из множества return удаленное число method add(self, int x): добавить число x в множество
