Подстрока в строке

Easy

Дана строка sss. Вам нужно написать класс SubstringFinder(s), а для него метод find(string t), который будет быстро определять, содержится ли строка ttt внутри строки sss.

class SubstringFinder: constructor(self, string s): создать находитель подстрок в s method find(self, string t) -> bool: return является ли t подстрокой s?
Разрешен только бор
Используйте только бор. Нельзя использовать хеширование, таблицы и прочие алгоритмы.
Проверка
Операция поиска подстроки в строке sss будет производиться qqq раз. Временная сложность нахождения подстроки ttt не должна быть кубической относительно qqq, ∣s∣|s|∣s∣ и ∣t∣|t|∣t∣.
