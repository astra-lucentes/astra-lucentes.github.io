Слишком большой массив

Easy

У меня есть волшебный компьютер и слишком большой отсортированный массив. Массив настолько большой, что в нем может содержаться несколько квинтиллионов элементов – до 101910^{19}1019. Мой волшебный компьютер может хранить столь большие объемы данных.

Вы должны написать функцию, которая сможет искать данный ей элемент в моем массиве. Если этот элемент есть в массиве, нужно вернуть его индекс. Если его нет, нужно вернуть none.

Функция должна выполнять поиск очень-очень быстро и никак не менять исходный массив (быть чистой).

Требуется

Написать чистую функцию

function search(array a, int x)

Функция ищет элемент x в массиве a. Если находит, то возвращает его индекс, иначе возвращает none.
